# 🎓 高考志愿填报助手 — 山东省版

> 3+3新高考 · 96个平行志愿 · "专业+学校"为志愿单位
> 初衷：同事的孩子有刚高考在准备填志愿的，希望能帮助他和其他的同学、家长更好地选择志愿。因为从我自己的经历而言，志愿填报影响了后来的人生路。专业选择对于人的影响太大了。

## 快速启动

```bash
cd backend

# 1. 安装依赖
pip install -r requirements.txt

# 2. 初始化数据（生成山东近3年模拟数据）
cd scripts && python load_data.py && cd ..

# 3. 启动后端
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 4. 打开浏览器访问
open http://localhost:8000
```

## 核心算法

### 1. 位次换算（一分一段表）

```
输入分数 → 查一分一段表 → 全省排名
```

### 2. 推荐引擎

```
加权预测位次 = 0.50×2025年位次 + 0.30×2024年位次 + 0.20×2023年位次

录取概率 = Φ((预测位次 - 我的位次) / σ)   # 正态CDF

策略分类:
  我的位次 / 预测位次 > 1.12 → 冲
  0.88 < 我的位次 / 预测位次 < 1.12 → 稳
  我的位次 / 预测位次 < 0.88 → 保
```

### 3. 蒙特卡洛模拟

```
对每个志愿: 实际录取位次 = 预测位次 + N(0, σ²)
模拟 10,000 次平行志愿录取过程
→ 滑档概率 + 安全指数 + 每个志愿命中概率
```

### 4. 志愿表策略评分

```
评分维度:
  - 冲稳保比例合理性 (15-25% / 40-50% / 25-35%)
  - 概率梯度是否递减
  - 保底是否足够（≥15个保底）
  - 蒙特卡洛滑档率
```

## 山东高考特点

- **3+3模式**：语数外(450分) + 6选3(300分) = 750分
- **不分文理**：统一划线，一段线约444分（本科线）
- **96个平行志愿**："专业(类)+学校"为1个志愿
- **选科要求**：物理/化学/生物/政治/历史/地理，不同专业有不同选科要求

## 当前数据范围（MVP）

- 50所院校（含省外985/211 + 省内重点）
- 17个专业（计算机/软件/通信/医学/法学/经济/会计/机械等）
- 93个"专业+学校"组合 × 3年(2023-2025) = 279条投档记录
- 一分一段表：3年 × 651个分数段
